IoT · Edge AI Team Project
Smart Guardian – AI Motorcycle Black-Box & ADAS

Engineered a distributed intelligent safety system for motorcycles that combines Advanced Driver Assistance Systems (ADAS) and a digital black-box recorder. The platform uses real-time computer vision, sensor fusion, and an Edge–Fog–Cloud architecture to detect hazards, warn riders proactively, and autonomously trigger emergency response pipelines during accident events.

Key Features

  • Real-time tailgating detection with custom-trained YOLOv8 model
  • Crash & rollover detection via Pixhawk IMU physics-based thresholds
  • Circular video buffer storing previous 10 seconds of evidence footage
  • Automated WhatsApp SOS alerts with live GPS location and video
  • Cloud incident upload pipeline to Supabase backend
  • Real-time buzzer warning system for proactive rider alerts

My Contributions

  • Configured Pixhawk telemetry via MAVLink protocol
  • Built ZeroMQ inter-device communication on Raspberry Pi
  • Developed crash-event handling and cloud connectivity logic
  • Conducted system integration, testing, and end-to-end validation
